Everything needed for the code is inside the zip file.

There is 2 independents components:
-the mainApp in the schedulers package
-the start in the GUI package
	use the GUI to setup the preferences needed to execute the whole application.
	
Steps for execution:
Start the GUI by starting the start class.
Go in option and set up the folder with you songs.Don't forget to save.
Now start the mainApp. Nothing visible will happen.
Reopen the GUI, you should see a list of events.
